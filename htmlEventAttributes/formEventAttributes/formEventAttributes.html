Form Event Attributes

onblur

This event fires the moment that the element loses focus. This is most often used with form validation code. (when a user leaves a form)

This event is the opposite of the onfocus event

The javascript function cannot be called "blur()". "blurr()" seemed to allow it to work.

Examples:

The text in the following input box will change to uppercase once the element loses focus.

Enter your name:

This is how its coded:

Syntax:

<element onblur="script">

Technical Details

Supported Html Tags : All HTML elements , EXCEPT: base, bdo, head, html, iframe, meta, param, script, style and title

onchange

This event fires the moment when the value of the element is changed

THis event is similar to oninput event, however the difference is that the oninput event occurs immediately after the value of an element has changed. While onchange occurs when the element loses focus the other difference is that the onchange event also works on select elements.

Examples:

This is how its coded:

Syntax:

<element onchange="script">

Technical Details

Supported Html Tags : "input type="textbox"", "input type="file"", "input type="password"", "input type="radio"", "input type="range"", "input type="search"", "input type="text", select and area

oninput

This event fires when an element gets user input

This attribute fires when the value of an input or textarea element is changed

THis is similar to the onchange event however the difference is that the oninput event occurs immediately after the value of an element has changed, while the onchange occurs when the element loses focus. The other difference is that the onchange also works on select elements.

Examples:

This is how its coded:

Syntax:

<element oninput="script">

Technical Details

Supported Html Tags : input type="password", input type="search", input type="text" and input type="textarea"

oninvalid

This event fires when a submittable input element is invalid.

For example, the input field is invalid if the required attribute is set and the field is empty. The required attribute specifies that the input field must be filled out before submitting the form.
