onmousedown, onmouseup, onclick, ondblclick

These events fire simply on the click or double click of the left mouse button on an element

The order of these events are:

  • onmousedown
  • onmouseup
  • onclick

    • A double click goes through 2 full cycles of single click events to be triggered and the single click events will be triggered in the process!

    A right click goes through the same events except an oncontextmenu is triggered after onmouseup.

    Examples:

    This is how its coded:

    Syntax:

    <element onmousedown="script">

    <element ondmouseup="script">

    <element onclick="script">

    <element ondblclick="script">

    Technical Details

    Supported Html Tags : All HTML elements, except: base, bdo, br, head, html, iframe, meta, param, script, style, and title

    onmouseover, onmousemove, onmouseout

    These events are triggered when the mouse pointer hovers over an element

    The events are triggered in the following cycle.

  • onmouseover - moves into the element
  • onmousemove - moves on the element
  • onmouseout - moves out of the element

    • Examples:

    This is how its coded:

    Syntax:

    <element onmouseover="script">

    <element onmousehover="script">

    <element onmouseout="script">

    Technical Details

    Supported Html Tags : All HTML elements, except: base, bdo, br, head, html, iframe, meta, param, script, style, and title

    onwheel

    This event fires when the wheel of the mouse is moved up or down when on an element.

    This also is triggered by the use of a touchpad zooming in.

    Examples:

    Use the scrollwheel whilst hovering over the image.

    This is how its coded:

    Syntax:

    <element onmousewheel="script">

    Technical Details

    Supported Html Tags : All HTML elements
