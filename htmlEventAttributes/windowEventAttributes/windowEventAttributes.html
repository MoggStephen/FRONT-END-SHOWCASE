Window Event Attributes

onafterprint

This works after after "ctrl p" is pressed and print has been selected to close the print dialog.

Syntax:

<element onafterprint="script">

Technical Details

Supported Html Tags : body

onbeforeprint

This works straight after "ctrl p" is pressed.

Syntax:

<element onbeforeprint="script">

Technical Details

Supported Html Tags : body

onbeforeunload

This event fires when the document is about to be unloaded. To get this one to work i had to use a slightly different script that is based at the top of the script element. This works to bring up the dilaog box to show that the event works.

Syntax:

<element onbeforeunload="script">

Technical Details

Supported Html Tags : body

onerror

This happens when an error occurs on loading an external file, which can be seen with the following image and in the console.

Syntax:

<element onerror="script">

Technical Details

Supported Html Tags : img, input(type image), object, link and script

onload

This event executes immediately after a page has been loaded which can be seen inthe console.

Syntax:

<element onload="script">

Technical Details

Supported Html Tags : body, frame, frameset, iframe, img, input(type image), link script and style

onoffline

This event is triggered when the connection ot the internet stops and the browser goes into work offline mode. This can be done in some browsers by choice.

Syntax:

<element onoffline="script">

Technical Details

Supported Html Tags : body

ononline

This event is executed when the browsers starts to work online from being offline.

Syntax:

<element ononline="script">

Technical Details

Supported Html Tags : body

onpageshow

This event occurs when a user navigates to another webpage. This event occurs everytime a page is loaded unlike the onload event which sometimes doesnt due to loading frmo cache.

Syntax:

<element onpageshow="script">

Technical Details

Supported Html Tags : body

onresize

This event fires when the browser window is resized

Syntax:

<element onresize="script">

Technical Details

Supported Html Tags : body

onunload

This event fires once a page has been unloaded or the browser window has been closed. It also occurs when navigating away from the page or submitting a form.

Syntax:

<element onunload="script">

Technical Details

Supported Html Tags : body
