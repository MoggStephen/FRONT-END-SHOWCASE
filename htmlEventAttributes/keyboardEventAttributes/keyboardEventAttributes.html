Keyboard Event Attributes

onkeydown, onkeypress, onkeyup

onkeydown

This event fires initally when the user is pressing a key on the keyboard

Syntax - <element onkeydown="script">

onkeypress

This event fires when the user presses a key on the keyboard

This event is not fired for all keys(ALT,CTRL,SHIFT,ESC) in all browsers. Use onkeydown to ensure whether the user has pressed a key.

Syntax - <element onkeypress="script">

onkeyup

This event fires when the user releases a key on the keyboard

Syntax - <element onkeyup="script">
