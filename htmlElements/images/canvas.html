Canvas - <canvas>

The canvas element is used to draw graphics on the fly using javascript usually.

The canvas tag is transparent and is only a container for graphics. You must use a script to actually draw the graphics

The text inside the canvas element will be displayed in browsers with javascript disabled and in browsers that do not support canvas.

It seems the script needs to take place after the canvas element unless the canvas element wont load.
