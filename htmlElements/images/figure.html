Figure - <figure>

The figure element specifies self contained content, like illustrations, diagrams, photos, code listings etc.

While the content of the figure elementis related to the main flow, its position is independant of the main flow and if removed it should not affect the flow of the document.

The figcaption element is used to add a caption of the figure element.
