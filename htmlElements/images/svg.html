Svg - <svg>

This element defines a container for SVG graphics.

This has several methods for drawing paths,circles,text and graphic images.

Svg has some predefined shape elements that can be used.

  • Rectangle
  • Circle
  • Ellipse
  • Line
  • Polyline
  • Polygon
  • Path
