img - <img>

This element is used to display a singular image.

Images are not technically inserted into a webpage. Images are linked to the webpage. The image tag creates a space for the referenced image.

There are 2 required attributes to the img tag.

The image tag requires the src and alt attributes.

You should always specify the height and width in the image tag. The page may flicker while the image loads.

This is probably down to the image element not creating the space for the referenced image, so when it does load it changes contents of the page.

srcset - Specifies a list of images files to use in different situations
