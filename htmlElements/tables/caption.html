Caption - <caption>

The caption defines a caption heading for inside of the table element.

This must be immediately after the table tag.

By default the caption is center aligned above the table, however the css properties text-align and caption-side can be used to align and place the caption.

A background colour on the table element does not affect the caption tag.


Examples:

Table 1

table caption at the top
row1 row2

Table 2

row1 row2
Caption at the bottom
