th - <th>

A th element defines a heading cell in an html table.

A html table has two kind of cells

  • Header cells which contain header information created with the th element
  • Data cells which contain data created with the td element

    • The text in td element are regular and left aligned by default, whereas th elements are bold and centered by default

    Attributes:

    The global attributes are supported. Global

    The event atttributes are suported. Event Attributes

    The colspan attribute is supported which defines a number of columns a cell should span. Example -> "colspan="2""

    The headers attribute is supported which allows you to link to the id of specific heading cells. Example -> "headers="name""

    The rowspan attribute is supported which defines a number of rows a cell should span. Example - > "rowspan="2""

    The abbr attribute is supported which specifies an abbreviated version of the content in the header. Example -> "abbr="car""

    The scope atribute is supported which specifies whether a header cell is a header for a column, row or group of columns and rows. Its value can be col, row, colgroup, rowgroup.
