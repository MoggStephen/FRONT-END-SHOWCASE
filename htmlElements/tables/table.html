Table - <table>

The table tag defines an HTML table on the webpage.

The table consists of the table tag and then one or more <tr>, <th> and <td> elements

tr defines a table row. th defines a table header, and the td element defines a table cell

A HTML table may also include <caption>, <colgroup>, <thead>, <tfoot>, and <tbody> elements
