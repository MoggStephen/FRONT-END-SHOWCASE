Colgroup - <colGroup>

This element allows a group of one or more columns in a table element for formatting

THe colgroup tag is used to apply styles to these selected columns instead of applying styles for individual cells.

The colgroup element should be a child of the table element and it should be after any caption element and before all the other elements found inside of a table element.

To define different properties to a column within the colgroup element use the col element


Examples:

heading 1 heading 2 heading 3 heading 4
cell 1 cell 2 cell 3 cell 4
cell 1 cell 2 cell 3 cell 4
