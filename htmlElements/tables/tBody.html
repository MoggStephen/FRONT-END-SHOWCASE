tbody - <tbody>

The tbody tag is used to group the body content together in a html table.

this element is similarly to the thead and tfoot elements to specify parts of the table.

Browsers use these elements to enable scrolling of the table body independently from the header and footer. Also when a table is printed these elements enable the footer and headers to be at the bottom and top of the page

The tbody element must have one or more tr tags inside of it

This element must be used as a child of the table element after any caption, colgroup, thead and tbody elements.

This element does not affect the layout of the table by default but can be styled with css.


Examples:
