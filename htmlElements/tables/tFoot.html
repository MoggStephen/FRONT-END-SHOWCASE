tfoot - <tfoot>

The tfoot element is used to group footer content in a html table.

The tfoot element isused in conjunction with the thead and tbody elements to specify each part of the table.

Browsers use these elements to enable scrolling of the table body independently from the header and footer. Also when a table is printed these elements enable the footer and headers to be at the bottom and top of the page

The tfoot element must have one or more tr tags inside of it

This element must be used as a child of the table element after any caption or colgroup elements and before any tbody tfoot and tr elements.

This element does not affect the layout of the table by default but can be styled with css.


Examples:

Table body cell 1 Table body cell 2 Table body cell 3
Table body cell 4 Table body cell 5 Table body cell 6
Table footer cell 1 Table footer cell 2 Table footer cell 3
