Head - <head>

The head element is mainly a container for metadata(data about data) and this data is not displayed on the page!

This is placed inbetween the html and body tag!

This data typically states the document title, character set, styles, scripts, and other meta information.

The following elements can go inside the head element:

  • <title>
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
