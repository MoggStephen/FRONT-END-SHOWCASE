Meta - <meta>

The meta element describes metadata (data about data) in the html document.

The meta tag has to be placed inside of a head element.

The meta tag is typically used to specify the following

  • Character set(this one needs to be done in every document)
  • Page description
  • Keywords
  • Author of the document
  • Viewport settings

    • This data is not displayed on the webpage but is machine readable

    Metadata is used by browsers in order to display content or reload pages

    Metadata is used by search engines to get the keywords associated with a webpage.

    Metadata on a whole is used by lost of web services


    View port manipulation

    There is a method that allows web designers to take control of the viewport. The view port is the visible area of a webpage.

    The viewport varies with the device and it will be smaller on mobile phones than on a computer display.

    It can be very usefull to add the following meta tag into all of your webpages in order to have a responsive web design.(suiting all sizes of screens)

    The width part of the contents sets the width of the page to follow the current devices screen width.

    The initial scale part sets the initial zoom level when the page is first loaded.

    Attributes:

    The global attribute is supported.

    The charset attribute is supported to specify the character endcoding for the document. This is always utf-8 for html5 documents.

    The content attribute is supported which is used to specify the text value associated with this meta tags use

    The http-equiv attribute is used to provide a HTTP header for the information/value of the content attribute. Allowed values are all HTTP headers

    The name attribute is used to specify a name for the metadata.
