Base - <base>

The base element is used to specify a base url for all relative URL's in a document

The base tag must contain a href or target attribute. Or both!

There can only be one single base element in a document, and importantly it must be in the head element!

Typically to best use this the base url should end with a / or the the url tag should start with a /.

A / gets added to the url when no / is present at the end of the base url or the start of the hyperlinks url.

An about blank blocked issue seems to arise when using the base url of 'https://'. I think this is because a base url of the starting url for all webpages is unsafe and immediately blocked?

The base tag can also be used to direct to a webpage too but the same premise of / use is required


Examples:

URL Hyperlinks:

|WORKS |METHOD |BASE URL |HYPERLINK URL |LINK|
|Correct! |Slash at end of base |https://www.coingecko.com/ |en |CoinGecko|
|Correct! |Slash at start of hyperlink |https://www.coingecko.com |/en |CoinGecko|
|Incorrect! |No slash at the base end or hyperlink start |https://www.coingecko |.com/en |CoinGecko|
|Incorrect! |base of https:// |https:// |www.coingecko.com/en |CoinGecko|

Default Styling:

None


Attributes:

The href attribute is used to supply the base url for all relative urls!

The target attribute is used to supply targets for the hyperlinks and forms in the page! _blank, _parent, _self, _top.

The global attributes are supported
