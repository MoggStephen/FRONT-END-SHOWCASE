Link - <link>

This element defines a relationship between the current document and an external resource to be used.

It is most often used to link to an external stylesheet or to add a favicon to a website

The link element is an empty element, it only contains attributes.

Attributes:

crossorigin - Specifies how the element handles cross origin requests.

href - specifies the location of the linked document.

hreflang -specifies the language of the text in the linked document.

media - specifies on what device the linked document will be displayed.

referrerpolicy - specifies which referrer to use when fetching the resource.

rel - This is required and specifies the relationship between the content and the linked document.

sizes -Specifies the size of the linked resource. this is only for "rel="icon""

title - defines a preffered or an alternate stylesheet.

type - specifies the media type of the linked document.

The global attributes are supported.

The event attributes are supported.
