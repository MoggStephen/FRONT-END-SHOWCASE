A - <a>

The a element defines a hyperlink which is used to link from one page to another.

The most important attribute of the a element is the href attribute. This indicates the links destination.

By default, links will appear as follows in all browsers:

  • An unvisited link is underlines and blue.
  • A visited link is underlined and purple
  • An active link is underlined and red.

    • If this element has no href attribute it will only be placeholder for a hyperlink. A linked page is usually displayed in the current window unless specified otherwise.

    Examples:

    Link to phone number

    Link to email address

    Open new window

    Attributes:

    download - This allows the target to be downloaded instead of taken to the target

    href - Specifies the url

    hreflang - specifies the language code of the linked document.

    media - specifies the media device linked document is optimized for.

    ping - specifies a space seperated list of urls to which when followed will post requests to be used for tracking.

    referrerpolicy - specifies the referrer information to send with the link.

    rel - specifies the relationship between the current document and the linked document.

    target - specifies where to target the linked document

    type - specifies the media type of the linked document.
