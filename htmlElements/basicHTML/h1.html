H1-H6 - <h1>-<h6>

These elements are tags to define html headings.

Obviously h1 defines more important headings and h6 defines the least important headings.

Use only one h1 per page. Try not to skip heading levels either as this can mess the structure of a webpage up.


Examples:

H2
