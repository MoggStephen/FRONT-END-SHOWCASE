BR - <br>

This element inserts a single line break

This is usefull for writing addresses or poems.

This tag is an empty tag which means it has no end tag.

This element should be used to enter line breaks, not to add space between paragraphs.


Examples:

Here is a br tag inbetween two block level elements(paragraphs):

Paragraph 1


Paragraph 2

Here is a br tag between to inline elements(span):
