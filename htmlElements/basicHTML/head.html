Head - <head>

The head element usually is a container for metadata. (data about data) and is placed between the html and body tag.

Metadata is data about the html document. Metadata is not displayed

This data typically defines the document title, character set, styles, scripts and other meta information

The following elements can go inside the head element.

  • title
  • style
  • base
  • link
  • meta
  • script
  • noscript
