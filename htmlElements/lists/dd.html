DD - <dd>

The dd element element is used to describe an individual term/name in a description list.

The dd element is used in conjunction with the dl element which describes a description list and dt which describes terms/ names

You can put paragraphs line breaks, images, links, lists in this element.


Examples:

Lexus
is200 silver
Vauxhall
Corsa Red
