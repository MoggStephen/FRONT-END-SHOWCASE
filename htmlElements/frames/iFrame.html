Iframe - <iframe>

This element specifies an inline frame

AN inline frame is used to embed another document withint the current html document.

Css can be used to style this frame. It is also best practice to give a title attribute for this element so screen readers can read out what the content is.

A title tag is best to be used so that screen readers can read out what the content is of the iframe is.

Other websites typically do not allow to be used within an iframe. This is due to a X-Frame-Option header of DENY or SAMEORIGIN which is added from the server side.

DENY means no one can load the website in iframe, SAMEORIGIN means only a page which is in the same domain can load the website.

Attributes:

allow - Specifies a feature policy for the iframe

allowfullscreen - Set to true if the iframe can activate fullscreen mode by calling the requestFullscreen() method

allowpaymentrequest - Set to true if a cross-origin iframe should be allowed to invoke the Payment Request API

height - Specifies the height of an iframe. Default height is 150 pixels

loading - Specifies whether a browser should load an iframe immediately or to defer loading of iframes until some conditions are met

name - Specifies the name of an iframe

reffererpolicy - Specifies which referrer information to send when fetching the iframe

sandbox - Enables an extra set of restrictions for the content in an iframe

src - Specifies the address of the document to embed in the iframe

srcdoc - Specifies the HTML content of the page to show in the iframe

width - Specifies the width of an iframe. Default width is 300 pixels
