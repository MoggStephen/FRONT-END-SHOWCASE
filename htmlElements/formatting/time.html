Time - <time>

This element defines a specific time.

The datetime attribute of this element is used to translate the time into a machine readable format so that the browsers can offer to add date reminders through the use of calendars. and search engines can produce smarter search results.


Examples:

Open from 10:00 to 21:00 every weekday.

I have a date on Valentines day.
