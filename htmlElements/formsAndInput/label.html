Label - <label>

The label element defines a label for the forms and input elements

Using labels benefits screen reader users so that the label will be read aloud appropriately.

Users who have difficulty clicking on very small regions such as checkboxes will be at aid as the the attached label element increases thje hitbox of the form element.
