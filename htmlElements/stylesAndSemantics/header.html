Header - <header>

The header element is a container for introductory content on a page.

It can also be used inside of an article element

It typically holds one or more h1-h6 elements, a logo icon and authorship information.

There can be multiple heading elements in any one document.

This element cannot be placed inside of footer, address or another heading element.

Header elements do not section content therefore it does not create a new section in the outline.
