Span - <span>

Span elements are most commonly used mid paragraph to colour part of text.

It is an inline container used to mark up a part of text or part of a document

This is easily styled by css or manipulated with javascript using the class or id attribute.

Span is much like the div element however div is a block level element unlike span which is an inline element

It can be used to group elements for styling purposes without line breaks and inline style(not taking up the rest of the width of the line)
