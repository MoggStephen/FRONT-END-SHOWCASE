Section - <section>

The section element defines a section within a document

This is typically used within the body element.

This is fairly similar to the main element however this element can be used as many times as needed.

A section typically contains headings and paragraphs

A section with no heading would not appear in the document outline. Creating a heading and making it hidden is how to solve it.

Secondary navigation is sometimes included in a section element.
