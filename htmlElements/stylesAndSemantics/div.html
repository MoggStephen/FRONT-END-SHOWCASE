Div - <div>

Div tags are used to divise a section or container in a html document

This section can then be styled or manipulated with css of javascript.

The class is easily styled using the class or id attribute

Any content can be put into the div tag

Div is a block-level container/element. It will take up the full width of the screen

Flexboxes are commonly being used with div elements.

CSS can be used to make elements that are similar to div elements.
