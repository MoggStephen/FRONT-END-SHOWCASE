Details - <details>

The details element encloses hidden information that the user can open on demand

An interactive widget is used to open and close the hidden content. By default the widget is closed. When opened the content is displayed

Any content can go inside the details tag

The summary tag is used alongside the details tag in order to supply a heading, as the default heading is "Details"

There is no built in way to animate the transition betweeen open and closed.

The triangular widget can be customised or hidden!
