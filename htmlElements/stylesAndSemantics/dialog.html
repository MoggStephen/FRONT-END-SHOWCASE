Dialog - <dialog>

The dialog tag defines a dialog box or subwindow

It is used to create popup dialogs and modals on webpages

The dialog element has similarities to the aside element and the details element

Attributes:

The dialog tag has an open attribute which is used to specifiy if the element is active and that the user can interact with it.

The global and event attributes are supported.
