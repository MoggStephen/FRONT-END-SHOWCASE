Main - <main>

The main tag specifies main content of a document.

The content inside of this element should be completely unique meaning contents shouldt be repeated in any other document.

Content like sidebars, navigations links, copyright information, site logos and search forms should not be included in this element.

There should only be one main element in any one document.

This element should not be contained in an article, aside, footer, header or nav element.

Typically main elements go inside of a body element.


Examples:

Content1

Content1 info

Content2

Content2 info


Default Styling:

None


Attributes:

The global and event attributes are supported.
